Array functions

  • myArray.length // Array length

  • myArray.push(item1, item2, ...) // add element to array

  • myArray.pop() // remove last element from array

  • myArray.shift() // get and remove Array first element

  • myArray.unshift(item) // add element in the start of Array

  • myArray.splice(index, [count], [replaceItem1], [replaceItem2], ... ) // remove current "index" element from array. You can tak count starts from index and new element for replacing removable elements.

  • myArray.concat(newArray) // returns two arrays concatenation

  • myArray.slice(index, count) // copying and return part of array and return it

  • myArray.join(summer) // returns array in one string joined using "summer" string

  • myArray.sort() // sorting array

  • myArray.reverse() // reversing array

  • myArray.indexOf(item) // returns first index of item

  • myArray.lastIndexOf(item) // returns last index of item

  • myArray.every(conditionFunction) // C# Linq All() method needs take argument condition Function

    • Example

    var numbers = [ 1, -12, 8, -4, 25, 42 ];
function condition(value, index, array) {
    var result = false;
    if (value > 0) {
        result = true;
    }
    return result;
};
var passed = numbers.every(condition);
document.write(passed); // false

  • myArray.filter(conditionFunction) // C# Linq Where() method needs take argument condition Function
  • myArray.forEach(someFunction) // C# Linq Foreach() method needs take argument some Function
  • myArray.map(someFunction) // this function as same as foreach function this function returns new array, but foreach not
