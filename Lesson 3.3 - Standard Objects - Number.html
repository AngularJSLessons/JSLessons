Standard Objects - Number

Number constructors

var x = new Number(34);

var y = new Number('34');

NUmber functions and constants

  • Number.MAX_VALUE - equals 1.79E+308

  • Number.MIN_VALUE - equals 5e-324

  • Number.NaN - Not a number

  • Number.NEGATIVE_INFINITY - return if number overflow limit

  • Number.POSITIVE_INFINITY - return if number overflow limit

  • isNaN - check variable is NaN or not

    • Number.isNaN - Not use this method Example : var f = Number.isNaN("hello"); // false

  • Number.parseInt(string) - parse string to Int
  • Number.parseFloat(string) - parse string to Float
  • Number.toFixed(decimalPlaces) - round Float number to decimal places
