String RegExp

Declare Regexp

var myExp = /hello/;
var myExp = new RegExp("hello");

RegExp functions

  • myExp.test(simpleText); // return true if text valid for RegExp
  • myExp.exec(simpleText); // возвращает ту часть строки, которая соответствует выражению. Если соответствий нет, то возвращается значение null

RegExp special symbols

  • [abc] - что строка должна иметь одну из трех букв.
  • [a-h] - строка должна содержать хотя бы один символ из диапазона.
  • [^a-h] [^abc] - ^ - строка имела только определенные символы
  • /[дт]о[нм]/ - "дом", "том", "дон", "тон"

RegExp properties

  • global(g) - позволяет найти все подстроки, которые соответствуют регулярному выражению. По умолчанию при поиске подстрок регулярное выражение выбирает первую попавшуюся подстроку из строки, которая соответствует выражению.
  • ignoreCase (i) - позволяет найти подстоки, которые соответствуют регулярному выражению, вне зависимости от регистра символов в строке.
  • multiLine (m) - позволяет найти подстроки, которые соответствуют регулярному выражению, в многострочном тексте.

Example

var initialText = "привет Мир";
var exp = /мир/; // var exp = /мир/i; Or var exp = /мир/ig;
var result = exp.test(initialText); // false
